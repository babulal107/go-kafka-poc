# go-kafka-poc
Learning Kafka integration with GoLang

### Producer
   Produce a message to Kafka Message Queue

### Consumer
   Read(Consume) a message from Kafka Queue 

## Step to run on local
  Run Kafka Queue Container:
```sh
    make run-kafka
```

Down Kafka Queue Container:
```sh
    make down-kafka
```

Run Consumer:
```sh
    make run-consumer
```

Run Producer:
```sh
    make run-producer
```
